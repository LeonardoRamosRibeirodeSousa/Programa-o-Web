A lista ordenada

Alista ordenada é representada pela tag <ol> e cada item da lista sera representado pela tag <li> Item List.Resultado:

  1. Item 1
  2. Item 2
  3. Item 3

Lista não Ordenada. - Unordered List

A Lista não Ordenada é representada pela tag <ul> e cada Item de Lista sera representado pela tag <li> Item List. Resultado:

  • Item 1
  • Item 2
  • Item 3
